
# Coordinated Narrative Detector — Full Implementation Blueprint

This subsystem identifies **multi‑platform, multi‑actor, synchronized messaging campaigns**, including:

- Foreign influence operations  
- Bot‑amplified narratives  
- Coordinated inauthentic behavior (CIB)  
- Cross‑platform message seeding  
- Hashtag brigading  
- Narrative laundering (OSINT → influencers → mainstream)  
- Timing‑based coordination  
- Persona‑network orchestration  

It fuses **linguistic**, **temporal**, **network**, and **behavioral** signals.

---

## 1. System Architecture Overview

### Core Components
- **Narrative Extraction Engine** — identifies emerging narratives across platforms  
- **Cross‑Platform Synchronization Engine** — detects timing‑based coordination  
- **Actor Network Analyzer** — maps relationships between accounts, personas, and bots  
- **Linguistic Similarity Engine** — identifies shared scripts, phrasing, and talking points  
- **Amplification Pattern Detector** — finds botnets, brigading, and mass‑engagement patterns  
- **Threat Scenario Engine** — models known influence‑operation TTPs  
- **Fusion Output Layer** — pushes correlated narrative events to dashboards & briefings

## 3. Narrative Extraction Engine

### Techniques
- Topic modeling (BERTopic, LDA, Top2Vec)  
- Transformer‑based clustering  
- Keyword & hashtag graphing  
- Named entity extraction  
- Narrative “fingerprinting”  

### Outputs
- Narrative clusters  
- Key phrases & talking points  
- Narrative evolution timeline  
- Platform‑specific variants  

---

## 4. Cross‑Platform Synchronization Engine

### Coordination Signals
- Near‑simultaneous posting across platforms  
- Identical phrasing within short time windows  
- Coordinated hashtag adoption  
- Synchronized engagement bursts  
- Multi‑platform seeding (Telegram → Twitter → TikTok)  

### Temporal Analysis
- Sliding windows: 5m, 30m, 2h, 24h  
- Burst detection  
- Lag correlation (who posts first, who amplifies)
